# 📣 UserStory-M1-W2 📣
A Python console program that manages multiple products through an interactive menu.
### ► What it does

  1. Shows a menu and waits for the user to pick an option.
  2. Validates the input — if invalid, shows an error and asks again.
  3. Lets the user add as many products as they want (name, price, quantity).
  4. Stores each product as a dictionary inside a list called inventory.
  5. Can display all registered products or calculate total stats.
  6. Exits only when the user chooses option 4.
